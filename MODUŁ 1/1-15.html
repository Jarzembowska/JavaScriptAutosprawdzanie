Rozgałęzienia (realizowane przez słowa kluczowe if...else) wykorzystywane są do wykonania innego kodu w zależności od warunków. Zasada działania jest prosta – wynik wyrażenia w warunku bloku if ewaluowany (obliczany) jest i sprawdza się czy jest true/truthy czy false/falsey. if (warunek) { // ciało if } else { // ciało else } Jeśli warunek sprowadza się do wartości true lub truthy, to wykonywany jest kod w nawiasie klamrowym następującym po warunku, a kod w bloku else zostaje zignorowany. let cost; const subscription = "pro"; if (subscription === "pro") { cost = 100; } else { cost = 0; } console.log(cost); // 100 Jeśli warunek sprowadza się do false/falsey, kod bloku if zostanie pominięty i wykonany zostanie kod w nawiasie klamrowym po bloku else. let cost; const subscription = "free"; if (subscription === "pro") { cost = 100; } else { cost = 0; } console.log(cost); // 0 ZADANIE Dodaj sprawdzenie pełnoletności użytkownika, używając parametru age do warunku w instrukcji if Jeśli użytkownik jest pełnoletni, powinien wykonywać się blok if i w zmiennej message zapisujemy string "You are an adult". W przeciwnym razie powinien wykonać się blok else i jako wartość zmiennej message zapisujemy "You are a minor". TESTY Zadeklarowane funkcję checkAge(age).
