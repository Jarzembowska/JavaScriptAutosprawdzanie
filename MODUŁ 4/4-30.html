METODA EVERY() Metoda every(callback) sprawdza, czy wszystkie elementy tablicy przechodzą test dostarczany przez funkcję callback. Zwraca true lub false. tablica.every((element, index, array) => { // Ciało funkcji callback }); Nie zmienia oryginalnej tablicy. Iteruje oryginalną tablicę element po elemencie. Zwraca true jeśli wszystkie elementy tablicy spełniają warunek. Zwraca false jeśli choć jeden element tablicy nie spełnia warunku. Iteracja tablicy zostaje przerwana, jeśli callback zwraca false. // Wszystkie elementy większe bądź równe zero? - tak [1, 2, 3, 4, 5].every((value) => value >= 0); // true // Wszystkie elementy większe bądź równe zero? - nie [1, 2, 3, -10, 4, 5].every((value) => value >= 0); // false Przy pracy z tablicą obiektów sprawdzana jest wartość jakiejś ich właściwości.
