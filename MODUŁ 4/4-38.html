METODA SORT() Metoda sort() sortuje elementy tablicy, ale w odróżnieniu od pozostałych iterujacych metod, robi to na oryginalnej tablicy. Sortuje i zmienia oryginalną tablicę. Zwraca zmienioną tablicę, to znaczy odnośnik do posortowanej oryginalnej tablicy. Domyślnie sortuje w porządku rosnącym. Sortowanie odbywa się drogą sprowadzenia wartości do łańcucha i porównania numerów porządkowych w tabeli Unicode. Taka tablica liczb zostanie posortowana w porządku rosnącym. const scores = [61, 19, 74, 35, 92, 56]; scores.sort(); console.log(scores); // [19, 35, 56, 61, 74, 92] Ponieważ domyślnie wartości sprowadza się do łańcucha, standardowe sortowanie liczb działa niezwyczajnie. Z tego względu w następnym zadaniu zobaczymy, jak wprowadzać swój porządek sortowania. const scores = [27, 2, 41, 4, 7, 3, 75]; scores.sort(); console.log(scores); // [2, 27, 3, 4, 41, 7, 75] Tablica łańcuchów sortuje się w porządku alfabetycznym. const students = ["Jacob", "Artemis", "Solomon", "Adrian", "Kai", "Ganymede"]; students.sort(); console.log(students); // [ "Adrian", "Artemis", "Ganymede", "Jacob", "Kai", "Solomon" ] Przy tym numer porządkowy małych liter jest mniejszy niż wielkich. const letters = ["b", "B", "a", "A", "c", "C"]; letters.sort(); console.log(letters); // ["A", "B", "C", "a", "b", "c"] Ponieważ sortuje się oryginalną tablicę, naruszona zostaje zasada czystości funkcji i nie można wygodnie utworzyć kilku pochodnych zbiorów na bazie oryginalnego. Na przykład, utworzyć zbioru posortowanego rosnąco, a drugiego malejąco. Dlatego przed sortowaniem tworzy się pełną kopię oryginalnej tablicy i ją się sortuje. const scores = [61, 19, 74, 35, 92, 56]; const ascendingScores = [...scores].sort(); console.log(scores); // [61, 19, 74, 35, 92, 56] console.log(ascendingScores); // [19, 35, 56, 61, 74, 92]
