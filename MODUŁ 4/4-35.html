METODA REDUCE() I TABLICA OBIEKTÓW Przy pracy z tablicą obiektów wykonuje się redukcja zgodnie z wartością jakiejś właściwości. Na przykład, istnieje tablica studentów z punktami za test. Trzeba otrzymać średnią ocenę. const students = [ { name: "Mango", score: 83 }, { name: "Poly", score: 59 }, { name: "Ajax", score: 37 }, { name: "Kiwi", score: 94 }, { name: "Houston", score: 64 }, ]; // Nazwa akumulatora może być dowolna, to po prostu parametr funkcji const totalScore = students.reduce((total, student) => { return total + student.score; }, 0); const averageScore = totalScore / students.length;
