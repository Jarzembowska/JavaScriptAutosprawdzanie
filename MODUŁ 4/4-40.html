WŁASNY PORZĄDEK SORTOWANIA ŁAŃCUCHÓW Do sortowania łańcuchów w porządku alfabetycznym, rosnąco lub malejąco, wykorzystuje się metodę łańcuchów localeCompare(). firstString.localeCompare(secondString) Jest wywoływana na łańcuchu, który trzeba porównać (firstString) z tym, co zostało do niego przekazane jako argument (secondString). "a".localeCompare("b"); // -1 "b".localeCompare("a"); // 1 "a".localeCompare("a"); // 0 "b".localeCompare("b"); // 0 Zwraca wartość negatywną jeśli firstString powinien być przed secondString. Zwraca pozytywną wartość większą niż zero, jeśli firstString powinien być po secondString. Jeśli łańcuchy są jednakowe, zwraca zero. Wygodnie jest stosować to podejście przy sortowaniu łańcuchów, ponieważ metoda sort() wymaga takich wartości od funkcji callback. const students = ["Jacob", "Artemis", "Solomon", "Adrian", "Kai", "Ganymede"]; const inAlphabetOrder = [...students].sort((a, b) => a.localeCompare(b)); console.log(inAlphabetOrder); // [ "Adrian", "Artemis", "Ganymede", "Jacob", "Kai", "Solomon" ] const inReversedOrder = [...students].sort((a, b) => b.localeCompare(a)); console.log(inReversedOrder); // [ "Solomon", "Kai", "Jacob", "Ganymede", "Artemis"
