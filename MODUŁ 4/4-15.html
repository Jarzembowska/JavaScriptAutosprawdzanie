METODA MAP() I TABLICA OBIEKTÓW Wiemy już, że zadania powszednie to manipulacja tablicą obiektów. Na przykład otrzymanie tablicy wartości właściwości ze wszystkich obiektów. Istnieje tablica studentów, a trzeba stworzyć oddzielną tablicę ich imion. const students = [ { name: "Mango", score: 83 }, { name: "Poly", score: 59 }, { name: "Ajax", score: 37 }, { name: "Kiwi", score: 94 }, { name: "Houston", score: 64 }, ]; const names = students.map(student => student.name); console.log(names); // ["Mango", "Poly", "Ajax", "Kiwi", "Houston"] Wykorzystując metodę map() można ziterować tablicę obiektów i w funkcji callback zwrócić wartość właściwości każdego z nich.
