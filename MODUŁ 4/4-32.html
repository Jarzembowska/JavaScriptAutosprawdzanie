METODA SOME() Metoda some(callback) sprawdza, czy choć jeden element tablicy przechodzi test dostarczony przez funkcję callback. Zwraca true lub false. tablica.some((element, index, array) => { // Callback function body }); Nie zmienia oryginalnej tablicy. Iteruje oryginalną tablicę element po elemencie. Zwraca true, jeśli choć jeden element tablicy spełnia warunek. Zwraca false, jeśli żaden element tablicy nie spełnia warunku. Iteracja tablicy zostaje przerwana, jeśli callback zwraca true. // Jest przynajmniej jeden element większy bądź równy zero? – tak [1, 2, 3, 4, 5].some(value => value >= 0); // true // Jest przynajmniej jeden element większy bądź równy zero? – tak [-7, -20, 3, -10, -14].some(value => value >= 0); // true // Jest przynajmniej jeden element mniejszy od zera? – nie [1, 2, 3, 4, 5].some(value => value < 0); // false // Jest przynajmniej jeden element mniejszy od zera? – tak [1, 2, 3, -10, 4, 5].some(value=> value < 0); // true
