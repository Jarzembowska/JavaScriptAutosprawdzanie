METODA FIND() Jeśli metoda filter(callback) wykorzystywana jest do wyszukiwania wszystkich elementów odpowiadającym warunkowi, to metoda filter(callback) pozwala znaleźć i zwrócić pierwszy pasujący element, po czym iteracja tablicy zostaje przerwana. Oznacza to, że szuka do pierwszego trafienia. массив.find((element, index, array) => { // Ciało funkcji callback }); Nie zmienia oryginalnej tablicy. Iteruje oryginalną tablicę element po elemencie. Zwraca pierwszy element odpowiadający warunkowi, to znaczy gdy callback zwraca true. Jeżeli żaden element nie pasuje, to znaczy dla wszystkich elementów callback zwrócił false, metoda zwraca undefined. Metoda find() jest wykorzystywana dla zadań – wyszukiwania elementu po unikalnej wartości właściwości. Na przykład, wyszukiwanie użytkownika po poczcie, samochodu po numerze rejestracyjnym, książki po tytule itp. const colorPickerOptions = [ { label: "red", color: "#F44336" }, { label: "green", color: "#4CAF50" }, { label: "blue", color: "#2196F3" }, { label: "pink", color: "#E91E63" }, { label: "indigo", color: "#3F51B5" }, ]; colorPickerOptions.find((option) => option.label === "blue"); // { label: "blue", color: "#2196F3" } colorPickerOptions.find((option) => option.label === "pink"); // { label: "pink", color: "#E91E63" } colorPickerOptions.find((option) => option.label === "white"); // undefined
