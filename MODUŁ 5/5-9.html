METODY KLASY Dla pracy z właściwościami przyszłego egzemplarza wykorzystywane są metody klasy. Metody to po prostu funkcje, z jednym wyjątkiem – dostępne są egzemplarzowi klasy. class User { constructor({ name, email }) { this.name = name; this.email = email; } // Метод getEmail getEmail() { return this.email; } // Метод changeEmail changeEmail(newEmail) { this.email = newEmail; } }
