OBIEKT PARAMETRÓW Klasa może przyjmować więcej danych wejściowych dla właściwości przyszłego obiektu. Jeżeli parametrów jest dużo (więcej niż 2), to zazwyczaj przyjmuje wzór "Obiekt parametrów". Idea tego wzoru leży w tym, aby przekazywać w postaci parametru jeden obiekt z logicznie nazwanymi właściwościami. Wartości właściwości takiego obiektu zamienią zbiór argumentów. class User { // Destrukturyzacja obiektu constructor({ name, email }) { this.name = name; this.email = email; } } const mango = new User({ name: "Mango", email: "mango@mail.com", }); console.log(mango); // { name: "Mango", email: "mango@mail.com" } const poly = new User({ name: "Poly", email: "poly@mail.com", }); console.log(poly); // { name: "Poly", email: "poly@mail.com" }
