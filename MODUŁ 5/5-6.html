DEKLAROWANIE KLASY Składnia literału obiektu pozwala utworzyć jeden obiekt. Często jednak potrzeba stworzyć wiele obiektów jednego typu z jednakowym zestawem właściwości, ale różnymi wartościami i metodami dla interakcji z nimi. Wszystko to należy zrobić dynamicznie, w czasie wykonywania programu. W tym celu wykorzystuje się klasy – specjalną składnię zadeklarowania funkcji dla tworzenia obiektów. Deklarowanie klasy zaczyna się od kluczowego słowa class, po którym następuje nazwa klasy i nawias klamrowy – jego ciało. Klasy przyjęło się nazywać, zaczynając wielką literą, a w samej nazwie odzwierciedlać typ tworzonego obiektu (rzeczownik). class User { // Ciało klasy } const mango = new User(); console.log(mango); // {} const poly = new User(); console.log(poly); // {} Rezultat wywołania new User() to obiekt, który nazwya się egzemplarz klasy, ponieważ zawiera dane i postępowanie opisywane przez klasę.
