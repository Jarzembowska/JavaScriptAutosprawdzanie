KONSTRUKTOR KLASY Dla inicjalizacji egzemplarza w klasie jest metoda constructor. Jeśli nie został zadeklarowany, tworzy się konstruktor domyślny – pusta funkcja, która nie zmienia egzemplarza. class User { // Składnia zadeklarowania metody klasy constructor(name, email) { // Inicjalizacja właściwości egzemplarza this.name = name; this.email = email; } } const mango = new User("Mango", "mango@mail.com"); console.log(mango); // { name: 'Mango', email: 'mango@mail.com' } const poly = new User("Poly", "poly@mail.com"); console.log(poly); // { name: 'Poly', email: 'poly@mail.com' } Wywołanie klasy z operatorem new prowadzi do stworzenia nowego obiektu i wywołania konstruktora w kontekście tego obiektu. To znaczy this wewnątrz konstruktora będzie odwoływać się do nowo utworzonego obiektu. To pozwala dodać do każdego obiektu właściwości z jednakowymi nazwami, ale różnymi wartościami. Właściwości name i email to właściwości publiczne, ponieważ będą własnymi właściwościami obiektu-egzmplarza i będzie można uzyskać do nich dostęp, zwracając się po kropce.
